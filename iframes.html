iframe Attributes
 iframes (inline frames) in HTML support several attributes that control their behavior and appearance. Some of these attributes include:
 1.allowfullscreen:This attribute allows the iframe to
  be displayed in fullscreen mode. It doesn't take a value; its presence alone enables fullscreen functionality.                    
  2. scrolling:This attribute controls the scrollbars
  within the iframe. It can take the values `yes`, `no`, or `auto`. However, it's worth noting that this attribute is deprecated in HTML5,and you should use CSS to control scrolling instead.
  3. sandbox:This attribute enables a set of extra restrictions  on the content hosted by the iframe. You can specify multiple restrictions by providing a space-separated list of tokens.
 Some of the common tokens include:
      allow-forms:  Allows the iframe to submit forms.
      allow-pointer-lock: Allows the iframe to use the Pointer Lock API.
      allow-popups: Allows the iframe to open pop-up windows.
      allow-same-origin:Allows the iframe content to be treated as being from the same origin, which is necessary for certain types of interactions.
      allow-scripts Allows the iframe to run scripts.
      These attributes help you control the security, usability, and appearance of iframes on your webpage.

Meta tag
viewportThe viewport meta tag in HTML is used to control the layout on mobile browsers. It ensures your web page is rendered correctly across different devices and screen sizes.
