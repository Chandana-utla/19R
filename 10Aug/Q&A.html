HTML Questions and Answers

1. What is the difference between <div> and <span>?
<div> (Division): Block-level element. Takes up the full width available, starting on a new line. Used for grouping large sections of HTML elements.
<span>: Inline element. Takes up only as much width as necessary, does not start on a new line. Used for grouping small chunks of text or elements.
2. What are the semantic elements in HTML?
Semantic elements clearly describe their meaning to both the browser and the developer. Examples include:
  • <header>
  • <nav>
  • <section>
  • <article>
  • <aside>
  • <footer>
  • <main>
3. What is the difference between href attribute and src attribute?
href (Hypertext Reference): Used in <a> (anchor) tags to link to other pages or resources.
src (Source): Used in elements like <img>, <script>, and <iframe> to embed external content.
4. What is the difference between XML and HTML?
XML (eXtensible Markup Language): Designed to store and transport data. Strict rules for closing tags, case sensitivity, and nesting. Does not define how data is displayed.
HTML (HyperText Markup Language): Designed to display data and structure web pages. Less strict, e.g., some tags can be left unclosed. Defines both the content and the layout of a webpage.
5. What are list tags and how many are they?
There are three main types of list tags in HTML:
  • Ordered List (<ol>): Creates a numbered list.
  • Unordered List (<ul>): Creates a bulleted list.
  • Description List (<dl>): Creates a list of terms with descriptions.

Git Commands Overview

August 12, 2024

1. What is the command for connecting a local repo and GitHub repo?
The command to connect a local repository to a GitHub repository is: 
git remote add origin <repository-URL>
Replace <repository-URL> with the URL of your GitHub repository.
2. Git command to change the directory of a local folder?
To change the directory in your terminal or command prompt, use the following command: 
cd /path/to/your/directory
Note: This is not a Git command but a basic command-line operation.
3. Git command to check the commits on the project?
To see a log of all the commits made to the project, use the following command: 
git log
4. Git command for pushing all the code to the ravi branch?
To push your code to the ravi branch, use the following command: 
git push origin ravi
5. Difference between the pull and push commands in Git?
git pull: Fetches updates from the remote repository and merges them into your local branch.
git push: Sends your local commits to the remote repository, updating it with your changes.
Key Difference: git pull brings changes from the remote repo to your local repo, while git push sends your changes from your local repo to the remote repo.
6. Command to check the number of branches and command to switch the branch?
Check branches: 
git branch
Switch to a branch: 
git checkout <branch-name>
Or, in newer Git versions: 
git switch <branch-name>
7. What are the stages of our project to reach the GitHub repo?
  • Working Directory: Where you make changes to files.
  • Staging Area (Index): Where you place files to be committed.
  • Local Repository: Where commits are stored on your local machine.
  • Remote Repository (GitHub): Where your commits are pushed and stored remotely.
8. The commands which we use in the staging area?
Add files to staging area: 
git add <file-name>
Add all files: 
git add .
Check status of the staging area: 
git status
